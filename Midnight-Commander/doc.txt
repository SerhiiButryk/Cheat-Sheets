# Explanation to the description of key combinations

Meta - "Alt", "Ctrl" or "Esc" key depending on OS (mac or linux) 

/**
 *
 *  File panel actions
 *
 */

Copy single: F5	

Move single: F6

Copy single in the current directory:	Shift + F5	

Move single	in the current directory: Shift + F6

/**
 *
 *  Search
 *
 */	    

Ctrl + s or Alt + s: Start a quick filename search in the directory listing 

+  - Select a group of files. (regular expression can be used)
\  - Unselect a group of files. (regular expression can be used)

Meta + Enter: Insert 'current object'1 to command line. 

Meta + t: Change panel view ('Full','Brief','Long') 

Meta + Shift + H: Display directory history.    

Ctrl + Space: Show directory size. 

/**
 *
 *  View panel modes
 *
 */

Meta + o: If the currently selected file is a directory, load that directory on the other panel and moves the selection to the next file. 

Meta + i: Make the current directory of the current panel also the current directory of the other panel.

Ctrl + U: Swap left and right panel.

Alt + p, Alt + n: Navigate between command line history.
